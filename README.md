## 使用 WebGL 在地图上绘制矢量图形：
###项目安装
```javascript
git clone https://github.com/xuff78/draw-on-map.git
cd draw-on-map
npm i
npm run dev
```
##### 基于 mapbox，使用 regl 构建的一套简易的 API，可绘制基本的矢量图形，可响应鼠标事件，功能待完善。欢迎指导

Mass：海量图标点显示。
